# ContactsManager

 Exercice 1 : Projet Java orienté objet.

## Description
Application de gestion de contacts en Java.
Elle permet d'ajouter des contacts et de les rechercher par nom.

## Structure du projet
- `Contact.java` : classe représentant un contact (nom + téléphone)
- `ContactsManager.java` : classe gérant le tableau de contacts
- `Main.java` : classe principale 

## Fonctionnalités
- Ajouter un contact
- Rechercher un contact par nom
- Afficher le numéro de téléphone d'un contact trouvé

## Exécution
Ouvrir le projet dans IntelliJ IDEA et lancer `Main.java`
